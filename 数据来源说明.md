# 项目财务评价与经济评价 - 数据来源说明

## 一、产品基本信息

### 1.1 产品定位
- **产品名称**：基于STM32的心率血压监测手表
- **开源项目地址**：https://oshwhub.com/NOZHIHUI/bi-she-ji-yustm32-di-xin-shuai-xie-ya-shou-biao
- **核心功能**：心率监测、血氧监测、步态/步数监测
- **主控芯片**：STM32F103C8T6（ARM Cortex-M3，32位MCU）
- **传感器**：MAX30102（心率/血氧）、MPU6050（六轴加速度计/陀螺仪）
- **开源协议**：GPL 3.0

### 1.2 技术架构（来自立创开源平台BOM和设计文档）
| 模块 | 元器件 | 功能说明 |
|------|--------|---------|
| 主控 | STM32F103C8T6 | 核心处理单元 |
| 心率/血氧传感器 | MAX30102 | PPG光电容积脉搏波测量 |
| 运动传感器 | MPU6050 | 步态检测、跌倒判断 |
| 显示 | OLED显示屏 | 数据显示 |
| 通信 | 蓝牙模块 | 与手机端数据传输 |

## 五、计算方法说明

### 5.1 净现值（NPV）
```
NPV = Σ(CF_t / (1 + r)^t)  其中 t = 0,1,2,3,4,5
CF_t = 第t年净现金流量
r = 折现率 = 12%
```

### 5.2 内部收益率（IRR）
```
Σ(CF_t / (1 + IRR)^t) = 0
通过fsolve数值求解
```

### 5.3 动态投资回收期
```
通过累计净现金流量现值曲线的二次拟合，求与x轴交点
```

### 5.4 盈亏平衡点
```
产量平衡点 = 固定成本 / (单价 - 单位变动成本)
利用率平衡点 = 产量平衡点 / 设计生产能力
单价平衡点 = (固定成本 + 变动成本总额) / 产量
```

### 5.5 敏感度系数
```
敏感度系数 = (IRR变化率) / (因素变化率)
临界值 = 因素变化率 / |IRR变化率| × |基准IRR - MARR|
```
